    Heartbleed vulnerability bulk checker tool supports custom ports other then 443 and Bulk IP/Domain checking .
    This tool can be used by pentesters to check the bulk IP with bulk custom ports which can be defined in ip.txt file and ports in port.txt file and then click heartbleedchecker.exe to list out the vulnerable systems to ListOfVulnerableIP_Domain.txt file as output.
    To execute the tool just double click on HeartBleedChecker.exe
