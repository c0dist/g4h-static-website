    Basic software required in backtrack or linux must for all
    when i was new to backtrack there was lot of problem like sound problem,irc , how to open tar files , screenshot ,and all blaaaa so i thought writing writing these post with all basic software reguired in backtrack or for another ubuntu flavour put sudo then apt-get


    Basic browser ==

    mozilla is the best browser along with it provide addon

    To install flash in the browser =
    apt-get install flashplugin-installer

    Also one good add on frequently used fast video downloaded (fvd)
    its very good download videos from you tube etc


    Email Client for the backtrack ==

    Evolution =

    is the email client

    to install it ===

    apt-get install evolution

    mozilla thunderbird =

    open source, cross-platform e-mail and news client developed by the Mozilla
    Foundation It can handle email, network news ,RSS feeds etc

    to install it ===

    apt-get install thunderbird


    Rss feeds =
    It is a Liferea is an abbreviation for Linux Feed Reader, a news aggregator
    for online news feeds. It supports the major feed formats including

    to install it ===

    apt-get install liferea



    Empathy ====

    Empathy is a messaging program which supports text, voice, and video chat and
    file transfers over many different protocols.Instant Messaging is one of the biggest
    ways for people to interact over the internet

    apt-get install empathy


    Internet relay chat ====

    XChat is popular irc used in linux

    apt-get install xchat


    gwibber Social Media Client
    Social media including twitter, facebook and more are now become more power full and successful
    so if you want use all these social networking website in a single place than get gwibber.

    apt-get install gwibber


    open office ====

    it just like office that we are using in windows operating system .
    apt-get install openoffice.org

    pdf ====

    to read pdf files

    apt-get install xpdf


    Movie/Video player ====
    totem media player
    it is the best player for Linux .

    apt-get install totem-gstreamer

    If you are experiencing sound and voice problem is backtrack
    than you must care about codec and other restrictions of Ubuntu.

    apt-get install ubuntu-restricted-extras
    apt-get install w32codecs
    apt-get install w32codecs libdvdcss2

    also to increase the volume yu can install

    alsamixer by installing

    apt-get install gnome-alsamixer


    video editing in linux ===
    apt-get install pitivi


    Rhythmbox ===
    Rhythmbox is a useful application that will play CDs if
    you insert an audio CD into your computer

    apt-get install rhythmbox



    extract the tar files =====
    ZIP ==

    UNZIP

    For tar.bz2,
    $ tar jxvf <filename.tar.bz2>

    For tar.gz,
    $ tar zxvf <filename.tar.gz>


    FTP Client for backtrack
    FTP (File transfer protocol) is an important protocol for transferring the large files,
    to install ftp if yu want then

    apt-get install gftp


    back track inbuilt firewall =====
    back | track has inbuilt firewall you enable or disable by following
    command in terminal
    ufw command == show diff command

    ufw status == firewall enable or disable

    For gui FIREWALL IN BACKTRACK ==
    apt-get install gufw

    TO TAKE SCREENSHOT ==
    apt-get install gnome-utils



    Software Manager for backtrack ===
    To get the right open source software if your Linux does not have software manager
    than you must install kate for it on the terminal.

    apt-get install kate


    Team viewer is the best software for desktop sharing and solving pc related problem

    download teamviewert 32 bit or 64 bit according to your backtrack from teamviewer.com

    dpkg -i teamviewer_linux.deb
