    From Noob to Ninja - FTW Web Skiddie Tutorial

    Step by Step:

    Noob steps:
    1. SQL Injection - Do a quot on the parameter, then do union select and play around the sql 'things'.
    2. Auth Bypass - Famous x=x method, needless to say, you how to do it.
    3. Brute Force - The script that you modify after looking at form tags and of-course your txt file (which Google helped to give you)

    For now, I will talk about the above 3 Only! Since, 99% of the people in n00b corner are limited but, am posting this to explore what is limited to you, properly.
    ================================================== =============
    Ninja steps:

    1. How does it work, in the inside? (ever thought???)

    id=8' - looks like it on the address bar but how does php/asp/jsp see it?

    PHP Code:
    $param = $_GET['id'];
    $sql = "select * from table where column = '$param'"; 
    $sql is a vulnerable step here.
    How?

    Normal case:
    "select * from table where column = '8'"

    Abnormal case:
    "select * from table where column = 8 union all select ............."

    and then, it is coded to ECHO things out which helps you fetch the information.

    ' <- this quote when added, it becomes: select * from table where column = '8''

    which results in an exception and the programmer who is foolish enough to code it without sanitizing inputs will also use mysql_error() which returns the text of error message.
    ================================================== ============

    2. Auth bypass - Everyone knows the 'string' but how does it work in the inside?

    "select * authorization where username = '$_POST['username']' AND password = '$_POST['password']";

    Now when you put the string using quote, notice there is a quote before $_POST['username'] and $_POST['password']

    after hitting the x=x string

    String which gets executed is:

    "select * from authorization where username = 'x' or 1=1 AND password = 'x' or 1=1";

    which returns boolean - TRUE! OR-ing in other words.

    ================================================== =============
    3. Brute Force - that py or pl file with .txt file next to it (in the same folder)

    Just as above, it uses AND until 200.

    ================================================== =============
    Real Ninja

    Did you just ask yourself what is 200? or you guessed 200 words in txt file?

    Google for 'HTTP status codes'.

    Explore.
    Analyze and learn.

    Don't learn to hack, hack to learn.
