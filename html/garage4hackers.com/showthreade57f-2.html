    Porting Vmware Image from product with more features to product with lesser features

    In an attempt to port a vmware image created on, let's say, ESXi server to a workstation or from a higher version of workstation to player, you might come across following error while power on the machine :

    Invalid configuration file. xxx was created by a VMware product with more features than this version of VMware Workstation and cannot be used with this version of VMware Workstation.
    Solution:
    Open .vmx file in some file editor and downgrade the entry for "virtualHW.version"

    e.g. if you ported from version 8 to version 7:
    change
    virtualHW.version = "8"

    to

    virtualHW.version = "7"
