    VIdigger-VMware Infrastructure Testing Tool
    As more and more companies are moving to adopt Virtualized systems,we as security consultants/auditors should be equipped to test the weakness of these systems

    I came across this tool which check the configuration of ESX server and the virtual machines hosted on ESX server against the VMware Infrastructure Hardening guide and other best practices.Though the core technology of VMware is designed to be secure, there are many configuration settings and deployment considerations that can severely impact the overall security of the deployed Virtual Infrastructure.

    VIDigger will retrieve configuration data about the host and virtual machines and then deliver a customizable report of problematic areas to the Virtual Administrator. VIDigger is intended to be used throughout the deployment lifecycle both as a hardening tool and a change tracking tool that brings configuration settings changes to the attention of the administrator. The tool focused on changes that may impact the overall security of the ESX host or virtual machine(s) deployed on the ESX host.
