    Opportunity at Qualys, Pune - WebApps Security Analyst/Penetration Tester
    As a Web Applications Security Analyst you will be responsible for ensuring the security of QualysGuard platform and applications. In this role you will be applying your web security and penetration testing experience to thoroughly analyze our QualysGuard cloud platform for vulnerabilities. You will be implementing various automated and manual techniques and detailed penetration testing procedures that will cover all aspects web security. You will work closely with our development teams to define the application security best practices, perform software architecture and design reviews, conduct black box/white box security testing, and support the identification, interpretation, and remediation of vulnerabilities across QualysGuard platform.



    Responsibilities

    Serve as the domain expert to assess web applications for security risks
    Periodic scanning (manual & automated) and security analysis QualysGuard platform to find web application vulnerabilities
    Lead ethical hacking and penetration testing sessions and remediation efforts
    Work with engineering teams during application development process to adopt secure design & coding practice
    Develop and extend tools that support investigation and improve product security posture
    Contribute towards increasing detection accuracy and efficiency of QualysGuard Web Application Scanner
    Collaborate with other security and product experts in Qualys to add new capabilities to QualysGuard product suite.



    Requirements

    Bachelors/Masters degree in Engineering, Information Security, Information Assurance, Network Security or related field
    5+ years of strong web application security experience with thorough understanding of web application vulnerabilities
    A passion for ethical hacking and vulnerability research
    Knowledge of secure coding principles and practices for web applications
    Hands-on experience with web application security assessment methodologies and tools

    Expert knowledge of information security principles, web applications and malicious code and common techniques used by hackers
    Knowledge of cloud-based infrastructures and how they affect security needs
    Experience with application security practices and methods such as OWASP Top Ten, WASC etc.
    Experience using vulnerability assessment scanners and source code analysis tools such as Burp Suite, Paros, Samurai WTF, and BackTrack
    Excellent problem solving and analytical skills; outstanding oral and written communication skills
    Self-motivation and the ability to work under minimal supervision are a must

    Programming knowledge using PHP, Java, Python, or Perl
    Strong knowledge of HTML, Javascript, CSS, XML, HTTP, HTTPS, SQL, TCP/IP
