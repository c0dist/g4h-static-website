    Accessing Services behind the NAT through HTTP/HTTPS
    Tunna is a set of tools which will wrap and tunnel any TCP communication over HTTP. It can be used to bypass network restrictions in fully firewalled environments.

    https://github.com/SECFORCE/Tunna

    Metasploit's portfwd does the same thing but for that to work you need to have a meterpreter session and I have run into environments where It wasn't available for me. This has helped on various ocassions. Thought I'd share this with ya.

    You do need a webshell on the server though in order for this to work.
