    Nothing better than learning something by breaking and rebuilding it. Learn what SQL injection is, how it happens, and how to protect your organization from it. SQL injections top plenty of lists as the most prevalent means of attacking front-end Web applications and back-end databases to compromise data. According to breach report released for 2011 a whopping 60 percent of all breach incidents examined involved SQL injections.

    Come join us in a hands on session and brainstorming where Amol explains how SQL Injection works, the different attack vectors, risks posed by SQL Injection, tools and techniques to master the art of Injection bug hunting

    1. LAMP Setup on linux (Apache/PHP/Mysql) OR XAMP/WAMP server on windows
    2. Running setup of Damn Vulnerable Web Application.
