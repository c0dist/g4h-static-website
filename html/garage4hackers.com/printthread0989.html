    Meld: Awesome GUI-diff tool for Linux
    Hello all,

    While working on a project, I felt the need for a diff tool for Linux, tried vim's but didn't like it. Searched for GUI counterpart and stumbled on this cool tool, meld. Works perfectly, at least did for me.

    Just install it using (tested on Ubuntu 12.04)
    Code:
    $ sudo apt-get install meld
    And launch it using
    Code:
    $ meld
    Hope this will help.

    Cheers. :)
