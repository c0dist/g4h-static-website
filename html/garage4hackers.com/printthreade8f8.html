    Finding imagebase in Ida disassembler
    I am aware of following 2 ways to find the imagebase of loaded executable in Ida disassembler:

    1. In Menu: Edit>Segments>rebase program. The address which is shown there is the image base address

    2. Run in Ida python console:
    Code:
    print "%x" % (idaapi.get_imagebase())
