    Profile visitor spam on Facebook unveiled

    Source


    Phishing and spamming are not new on commercial as well as social networking site. Be it Facebook or Orkut, none have been left untouched by hackers and spammers. Orkut a suffered a lot last year due to a malware broke out which had infected thousands of user profile. Rogue Facebook applications and features has always been in news from past few months. While browsing Facebook, we saw a new spam spreading widely known as “Who visited your profile” or “facebook profile viewer” . We pointed out event notification as well as post on walls:



    It looks tempting to users who are unknown what this will result in. We checked the link (http://x.co/WlL0/?96788) mentioned in the profile visitor. It landed us to the following page :



    This was just a URL shortner link shortened by http://app.x.co/ which landed us to a blog on blogs-pot. We can see a script on that page which the spammer claims to be a profile peeker code. We observed a URL in the code ://bbindi4.in/jsp.php and opened it. We checked the source code of the page and noticed another URL in it:



    The JavaScript redirects us to the page mentioned in the source code. It takes us to another blog-spot blog asking us to click on an ad or complete a survey:



    So its very clear now that there is nothing such called as “Facebook profile viewer” or so. This is just a plain social engineering used by hackers to spread spam and earn some bugs. The conclusion is simple. Hackers are trying to redirect users to their blog-spot pages and earn bugs by making them click on ads sponsored by Google ad sense. We advice readers not to click on these links. Any new feature added by Facebook is updated time to time. Be safe and Be secure
