    Windbg Narly
    Narly is an extension for Windbg which helps us in fetching details about the Loaded Modules of an Executable such as following:

    • /SafeSEH protected
    • DEP and ASLR compatible modules
    • /GS flag set


    Link: http://code.google.com/p/narly/

    easy to use:

    .load narly to load the windbg extension

    and

    !nmod to run it.

    I have tested this on Windows 7 and it works good.
