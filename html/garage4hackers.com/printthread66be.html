    Here are the few things that I'd like to contribute/add/suggest to the video:

    1. Can we have no songs in the background from the next video, plz? Songs are kinda distracting, at least to me.

    2. While you were demonstrating the slicing of strings, i.e s[0:5], etc, the correct word must be "index" instead of "sequence".

    3. Kudos for showing the use of "dir( )". This is actually one of the things that helped in learning Python.

    4. Strings are covered some more in awesome Python documentation. I'd suggest you can put this in your video's description -> https://docs.python.org/2/tutorial/i...n.html#strings

    5. The help feature that you demonstrated in video is feature provided by IDLE, not a standard Python feature. What you can use instead is "help( )". Example.
    Code:
    >>> help(str.replace)
    Help on method_descriptor:

    replace(...)
        S.replace(old, new[, count]) -> string
       
        Return a copy of string S with all occurrences of substring
        old replaced by new.  If the optional argument count is
        given, only the first count occurrences are replaced.
    This works everywhere, regardless of platforms, IDEs, etc. Another helpful feature I use all the time. :)
